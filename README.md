# 에이전트로 쌀먹하기: AI Transformation으로 실무 효율화하기

> **사이프(SIPE) 5기 | 1차 미션**

## 미션 주제

반복되는 업무를 AI 에이전트 팀으로 어디까지 자동화할 수 있는지 직접 실험하고, 실제 사내 업무에도 적용 가능한 수준까지 발전시켜봅시다.

## 미션 목적

이 미션은 **AI Transformation** 관점에서 실제 개발/업무 환경의 반복 작업을 분석하고, 이를 보조하거나 자동화할 수 있는 **AI 에이전트 팀**을 직접 설계·구현해보는 것을 목표로 합니다.

단순히 챗봇 하나를 만드는 것이 아니라,

**기획 → 개발 → 테스트 → 문서화 → 배포/운영**으로 이어지는 실제 실무 흐름 안에서 각 역할에 맞는 에이전트를 만들고, 이를 유기적으로 연결해보며 실무 생산성을 높일 수 있는 협업 방식과 자동화 구조를 직접 검증해보려 합니다.

## 에이전트 예시

| 에이전트                               | 설명                                      |
| -------------------------------------- | ----------------------------------------- |
| 요구사항 정리 에이전트                 | 요구사항을 구조화하고 정리                |
| 코드 생성 / 리팩토링 보조 에이전트     | 코드 작성 및 개선 보조                    |
| 테스트 코드 작성 에이전트              | 테스트 코드 자동 생성                     |
| PR 리뷰 보조 에이전트                  | Pull Request 코드 리뷰 보조               |
| 문서화 에이전트                        | 코드 및 프로젝트 문서 자동 생성           |
| 장애 분석 / 로그 요약 에이전트         | 장애 로그 분석 및 요약                    |
| 배포 체크리스트 점검 에이전트          | 배포 전 체크리스트 자동 점검              |
| 운영 이슈 분류 및 대응 가이드 에이전트 | 운영 이슈 자동 분류 및 대응 가이드 제공   |
| 비즈니스 구조 개선 에이전트            | 사내 비효율적인 비즈니스 구조를 AI로 대체 |

## 미션 대상

- AI를 단순히 체험하는 수준이 아니라, 실제 서비스나 업무 플로우에 녹여보고 싶은 분
- 실무 개발 생산성 개선에 관심 있는 분
- 에이전트 / 자동화 / LLM Ops를 함께 실험해보고 싶은 분
- 단순 데모가 아니라 현업에 적용 가능한 결과물을 만들어보고 싶은 분
- AI 경험이 많지 않더라도 문제 정의, 설계, 구현, 테스트, 문서화 등 다양한 방식으로 기여해보고 싶은 분

## 주차별 계획

| 주차       | 내용                                             |
| ---------- | ------------------------------------------------ |
| 1주차      | 비효율적인 실무 업무 사례 수집 및 문제 정의      |
| 2주차      | 에이전트 역할 분리, 워크플로우 및 기술 스택 설계 |
| 3주차      | 핵심 에이전트 1~2개 구현 및 기본 동작 검증       |
| 4주차      | 멀티 에이전트 연결 및 외부 도구 연동             |
| 5주차 이후 | 실무 시나리오 기반 테스트, 개선, 데모, 회고      |
