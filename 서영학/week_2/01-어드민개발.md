# 1. 어드민 개발 

## AI Agent 병렬 활용 
- 페이지별 개발 병렬 실행을 위한 브랜치 분류 작업 
- develop -> 1차 브랜치 (merge 후 확인용) -> 작업별 worktree 분류 및 agent 세션 실행
- 각 AI 작업이 완료되면 우선 코드 확인 후 1차 브랜치에 merge 후 검증 작업
- 모든 작업이 완료되면 1차 브랜치에서 develop에 pr (squash)

## 어드민 관련 AI Agent에게 주어놓은 스킬 및 Agent.md 
- 로케일 등 공통 변수 사용 가이드
- 버튼 설계 (검증 단계를 포함한 구조) 및 로직 주입 형태 가이드 
- 테이블 설계, 필터와 선택 
- 입력시, 파일명의 패턴에 따라 자동 선택 지원하도록 가이드
